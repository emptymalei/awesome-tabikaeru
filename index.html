旅行青蛙明信片 GitHub Source Code
这只是自己玩游戏过程中呱蛙儿寄回来的明信片，数量非常有限。如果你也想分享你的明信片，在这里 fork, pull request 就好了。所有的明信片放在了 "postcards" 文件夹下面，命名方式是 "tabikaeru-[scene]-[friends].png"，其中如果是独自一人的话， "[friends]" 就省略了。例如独自在树枝上就命名为 "tabikaeru-branch.png"，如果是跟蝴蝶在树枝上就命名为 "tabikaeru-branch-butterfly.png".
